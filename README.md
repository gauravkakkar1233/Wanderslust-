# Wanderlust 🌍

A full-stack Airbnb-like listings web application built using the MERN stack.

## 🚀 Features
- Create, edit, delete listings
- Server-side validation using Joi
- Image URL handling with defaults
- RESTful routing
- Error handling with custom middleware

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS & Bootstrap
- Joi (validation)

## 📦 Installation
```bash
git clone https://github.com/gauravkakkar1233/Wanderslust-.git
cd Wanderslust-
npm install
nodemon app.js

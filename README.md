An Airbnb-inspired full-stack web application where users can explore, create, edit, and manage travel listings with a clean and responsive interface.

---

## 🚀 Live Demo

🔗 GitHub Repository:  
https://github.com/gauravkakkar1233/Wanderslust-

---

## ✨ Features

- 🏡 Create, edit, and delete listings
- ✅ Server-side validation using Joi
- 🖼️ Image URL handling with fallback images
- 🔁 RESTful routing
- ⚠️ Custom error handling middleware
- 🎨 Responsive UI using Bootstrap
- 📂 MVC project structure

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/gauravkakkar1233/Wanderslust-.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd Wanderslust-
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Development Server

```bash
nodemon app.js
```

---

## 🌐 Environment Variables

Create a `.env` file in the root directory and add the following:

```env
ATLASDB_URL=your_mongodb_connection_string
SECRET=your_secret_key
CLOUD_NAME=your_cloudinary_cloud_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret
```
